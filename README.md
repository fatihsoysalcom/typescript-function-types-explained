# TypeScript Function Types Explained

This example demonstrates the core concepts of TypeScript function types. It shows how to declare function types, implement functions matching those types, and leverage type safety to prevent common JavaScript errors at compile time.

## Language

`javascript`

## How to Run

1. Save the code as `function_types.ts`.
2. Compile it using the TypeScript compiler: `tsc function_types.ts`.
3. Run the generated JavaScript file: `node function_types.js`.
